# 🏀 Basketball Math Trainer - Backend

Backend עם Gemini Vision AI לאפליקציית תרגול מתמטיקה לכיתה ד'.

## הפעלה מקומית

```bash
npm install
npm start
```

השרת ירוץ ב-`http://localhost:3000`

## משתני סביבה

יש להגדיר ב-`.env`:
- `GEMINI_API_KEY` - מפתח API מ-Google AI Studio
- `PORT` - אופציונלי (ברירת מחדל: 3000)

## Deploy

הפרויקט מותאם ל-Render.com / Railway / Fly.io
